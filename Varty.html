Pembahasan
Jawaban: D

Langkah-langkah merupakan salah satu bagian dari struktur teks prosedur yang merupakan cara yang ditempuh agar tujuan itu tercapai, dan bagian langkah ini merupakan urutan yang tidak dapat diubah langkahnya sehingga ketika seseorang melakukan aktivitas atau kegiatan secara benar.

Berdasarkan penjelasan di atas, maka bagian inti dari langkah pembibitan tanaman adalah langkah nomor:

  • (2) Membuat bedeng persemaian.
  • (3) Memberi atap tempat persemaian.
  • (4) Menyemai benih.
Langkah nomor 2, 3, dan 4 tersebut merupakan bagian inti dari langkah pembibitan tanaman karena tanpa langkah tersebut tidak akan tercapai tujuan yang diinginkan yaitu menghasilkan bibit tanaman.

Contoh Soal 3

Perhatikan drama singkat berikut ini!

Naina : Hai sandrina, apakah kamu mengikuti program magister di UI?
Sandrina : Iya Naina, kenapa memangnya?
Naina : Saya sudah mendaftar, hanya saja bingung bagaimana cara membayarnya.
Sandrina: Bayar di ATM lebih simpel dan tidak usah mengantri.
Naina : Oh bisa ya Sandrina ?
Sandrina : Iya sangat mudah, cara pembayarannya kamu harus masuk ke menu transaksi lainnya di layar ATM, kemudian pilih menu pembayaran, pilih menu pendidikan, setelah itu masukkan nomor pelanggan.
Naina : Maksud dari nomor pelanggan itu apa?
Sandrina: Masukkan kode nomor registrasi dan institusi. Contohnya, UI memiliki kode 050 dan setelah itu masukkan nomor registrasi yang kami miliki. Setelah itu, ikuti petunjuknya saja, mudah kan?

Judul teks prosedur kompleks di atas adalah:

  • Tata cara mendaftar program magister UI
  • Prosedur registrasi ujian
  • Tata cara bagaimana membuka rekening di bank.
  • Tata cara mencairkan dana di bank.
  • Prosedur pembayaran registrasi program magister UI via ATM
Pembahasan

Jawaban: C

terdapat tujuan dan langkah-langkah yang muncul pada teks drama. Tujuan sangat kompleks berada di dialog ketiga tuturan sandrina yaitu mudahnya melakukan pembayaran registrasi di UI via ATM.
Berdasarkan paparan tersebut, maka jawaban yang paling tepat adalah C yaitu prosedur pembayaran registrasi program magister UI via ATM.
